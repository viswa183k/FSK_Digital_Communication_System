# FSK Digital Communication System

This project demonstrates Frequency Shift Keying (FSK), a digital modulation technique where binary data is transmitted using two distinct frequencies. The system generates different tones for binary 0 and 1, simulating real communication systems.

## Features
- Binary data transmission using frequency modulation  
- Demonstrates digital communication principles  
- Simple and effective modulation technique  
- Real-time signal generation  

## Hardware Required
- Arduino board  
- Buzzer or speaker  
- Connecting wires  

## Applications
- Wireless communication systems  
- Data transmission protocols  
- Embedded communication projects
